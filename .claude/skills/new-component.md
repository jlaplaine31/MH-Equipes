# Nouveau composant

Cree un composant UI reutilisable avec Tailwind CSS.

## Etapes

1. Demander le nom et l'usage du composant
2. Determiner s'il va dans `src/components/ui/` (generique) ou `src/components/teams/` (specifique)
3. Creer le fichier `{ComponentName}.tsx` avec les props typees
4. Utiliser Tailwind pour le styling
5. Supporter le theme clair/sombre via les variables CSS ou les classes Tailwind dark:
